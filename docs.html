the complete guide

a digital being with a fixed birth and death date. it owns a Solana wallet that bleeds value daily. four on-chain actions keep it alive.

vital stats

  • 💵 bag · 😼 vibe · ⚡ grind · 🐅 pride
  • decay every 10 s, floor 20, cap 100
  • each on-chain action restores +10 to one stat

actions

each button sends 0.0001 SOL to the wallet with a Memo instruction tagging the action: feed (💵 bag), hug (😼 vibe), play (⚡ grind), talk (🐅 pride). one action per wallet per UTC day.

cost curve

day 1 = $1, day 100 = $100, day 3650 = $3,650 (in SOL, via CoinGecko). when the wallet drops below the daily cost, it dies broke.

clocks

  • mega clock: real elapsed since birth
  • human clock: real elapsed (in cabinet)
  • x10 clock: 10× elapsed — 1 human day = 10 inner days

trollbox

P2P chat over Gun.js. anonymous name auto-generated; replaced with your truncated wallet address when connected.

tech

static HTML/CSS/JS · Solana mainnet · @solana/web3.js v1.98 (esm.sh)
Phantom + Solflare · Memo Program · Helius/RPC · CoinGecko · Gun.js · localStorage

configure

edit js/config.js to set mascot name, birth date, wallet, RPC URL.
